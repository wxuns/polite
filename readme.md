**Polite做了那些事**
----
1. 一个基于yaf的PHP框架
2. 支持composer
3. 类似于Laravel的命令行工具
4. 数据库迁移
5. 集成Seaslog高性能日志扩展
6.  ...

**扩展要求**
----
编译安装yaf、seaslog扩展

**安装步骤**
----
~~polite目前还处于测试开发中，不建议生产环境中使用~~
```php
composer create-project wxuns/polite
git clone https://github.com/wxuns/polite.git
```
